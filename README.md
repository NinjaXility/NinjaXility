# Tools that stop when you close them.

Local-first. No accounts. No tracking. No cloud.

**Built for use, not growth.**

I choose restraint over reach,  
clarity over scale,  
utility over everything else.

If it's here, it solved a problem once  
and didn't need to become anything more.

> Tools don’t need an audience.  
> They just need to work.  
> Building quietly.
